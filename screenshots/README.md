# Screenshots

Place your screenshots here before publishing:

1. `demo_layout.png` — Run the app and capture the full IDE-like layout
2. `drag_drop.png` — Capture while dragging a tab (showing the blue drop indicator)

## Requirements for pub.dev

- File types: png, jpg, gif, or webp
- Max 4 MB per image
- Recommended: ~800-1200px wide for good display on pub.dev
- Animated GIFs are allowed (great for showing drag interactions)

## How to capture

```bash
# Run the demo app
flutter run -d windows

# Use Windows Snipping Tool (Win+Shift+S) to capture
# Or use any screen recording tool for GIFs
```
